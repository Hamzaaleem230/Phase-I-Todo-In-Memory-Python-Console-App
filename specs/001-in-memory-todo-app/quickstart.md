# Quickstart: In-Memory Todo Console App

This guide explains how to run the application.

## Prerequisites

- Python 3.13 or higher.

## Running the Application

1.  Navigate to the root directory of the project.
2.  Run the application using the following command:

    ```bash
    python main.py
    ```

3.  The application will start, and you will see a menu of options to choose from.
